# 보다(BODA)

보험증권과 보험약관을 업로드하면, AI가 분석해서 **청구 가능 여부 · 예상 보험금 · 필요 서류 · 보장 항목**을 알려주는 보험 분석 서비스입니다.

## 주요 기능

- **4가지 질문 흐름**
  1. 청구 가능한지 먼저 알고 싶어요
  2. 예상 보험금을 먼저 알고 싶어요
  3. 필요 서류를 먼저 알고 싶어요
  4. 내 보험의 보장 항목부터 보고 싶어요
- 보험조건 입력 모달을 통한 상세 조건(발생 상황, 치료 종류, 입원/깁스/치아 정보 등) 입력
- 보험증권 · 보험약관 PDF 업로드 및 분석 상태 폴링
- 증권 최대 3개까지 업로드 가능 (보장 항목 합산 대시보드)
- 결과 화면에서 약관 근거(citation) 확인
- 다른 질문(칩1~4)으로 이어서 확인하는 후속 질문 기능
- 자유 입력으로 추가 질문을 이어가는 멀티턴 채팅
- 파일 형식/크기/개수 검증 및 에러 안내

## 기술 스택

- React
- React Router
- Vite

## 프로젝트 구조

```
src/
├── api/          # 백엔드 API 호출 (chat, upload, dashboard, evidence)
├── components/   # 공용 컴포넌트 (모달, 카드, 네브바 등)
├── pages/        # 라우트별 페이지
├── utils/        # 응답 데이터 매핑 등 유틸
└── data/         # 프리뷰용 샘플 응답 데이터
```

## 실행 방법

```bash
npm install
npm run dev
```

## 참고

- `/result/preview/:sampleKey` 경로로, 실제 업로드 없이 저장된 샘플 응답으로 결과 화면을 바로 확인할 수 있습니다.
